## Informações do Projeto

Prompt de comando utilizado no Gemini

 Tema: Planilha de gastos com gráficos interativos para monitorar os tipos de gastos

Objetivo
Muitas pessoas não sabem ou não têm a informação necessária de como distribuir sua renda com a quantidade de gastos realizados, ou seja, pouco sabem sobre educação financeira. 
O objetivo é ter uma IA que ajuda com o controle de gastos, separando-os em tipos de gastos para melhor gerenciamento do dinheiro. A IA deve mostrar de forma visual de forma a facilitar o entendimento do usuário. A ferramenta deve possibilitar a conexão com bancos para ser preenchida de forma automática, caso seja de interesse do usuário.
Público alvo: Pessoas com pouco conhecimento em gerenciamento de gastos e pessoas que necessitem da ferramenta para auxiliar na monitoração mensal das contas pessoais.

Problema: A falta de controle ou acompanhamento das finanças pessoais podem gerar um acúmulo de dívidas e fazer com que uma pessoa fique refém de empréstimos e renegociações das mesmas, gerando não só problemas financeiros, mas também problemas pessoais, de relacionamento e mentais. Dessa forma, se faz necessário um sistema acessível e de uso facilitado para auxiliar essas pessoas.

Hipóteses: o usuário fazendo o uso da planilha, promoverá atenção aos gastos extras e melhor controle financeiro, além de identificar os maiores tipos de gastos e reduzir o tempo caso a planilha fosse feita de forma manual.

Restrições: a ferramenta deve funcionar de forma gratuita, com segurança sobre os dados dos usuários, sem que haja coleta de dados sensíveis para o seu funcionamento.
